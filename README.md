# Laravel Basics Collection

This folder contains a set of small Laravel projects created for learning and practice. Each subfolder focuses on one topic, so the repository works like a portfolio of Laravel fundamentals.

## Projects

| Project | Summary |
| --- | --- |
| MyAuthentication | User registration, login, and session-based authentication with protected dashboard access. |
| myca | A small college website that demonstrates routes, named routes, and Blade-based page navigation. |
| MyControllers | Form handling and validation with two styles of error display: global and inline. |
| MyEmail | Email sending examples using Laravel's mailing features. |
| MyExam | Student exam management with data insertion, tabular output, and result tracking. |
| MyFirstProject | Introductory routing project showing strings, HTML, JSON, downloads, and redirects. |
| MyFormsValidations | Form validation examples using Laravel rules and custom validation logic. |
| MyMiddleware | Authentication flow protected by middleware-based route access control. |
| MyMongodb | Student CRUD operations using MongoDB as the database backend. |
| MyMultipleUpload | Multiple image upload handling and file submission workflow. |
| MyQueryBuilder | User database operations using Laravel Query Builder with language options. |
| MySession | Session storage, retrieval, and deletion examples. |
| MyUploadFile | Basic file upload and display workflow. |

## Notes

- Root-level `node_modules` and `vendor` folders are ignored because they are generated dependencies.
- The `Xampp_instruction_confg.md` file is treated as a local setup note and is not meant for GitHub.
- Each subproject can be opened independently and run with its own Laravel setup.
